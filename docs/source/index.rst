Getting Started
================

This section will provide an introduction and some tips to operate TomoScan BEATS DAQ.


**Introduction**


.. admonition:: Tips 
    
    Before starting the operation, make sure to check the following:
        1. Check the **Operation Console** is running.
        2. Check the **DAQ Server** is running.
        3. Check the **Writer Server** is running.
        4. Check the **Motor Driver Unit** is running.
        5. Check the **Camera** is running.
        6. Check the **Air conditioner** is ON.
